# Portfolio Project

A modern, responsive developer portfolio website showcasing projects, work experience, and technical skills.

---

## 📁 Project Structure
portfolio-replica/
├── index.html # Main HTML file
├── style.css # Styles & responsive design
├── script.js # JavaScript functionality
└── public/ # Images, logos, and assets


---

## 🚀 Quick Start

1. Download or clone the repository
2. Open `index.html` in your browser
3. That's it — no build steps or dependencies

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| Smooth scrolling | Anchor links scroll gently to sections |
| Active nav links | Highlights current section on scroll |
| Back to top button | Appears after scrolling down |
| Responsive layout | Works on mobile, tablet, and desktop |
| Project cards | Clickable cards with hover effects |
| Work experience | Job history with company logos |

## 📱 Responsive Breakpoints

- **1024px** – Padding adjustments
- **820px** – Single column projects
- **640px** – Stacked navigation
- **520px** – Full width buttons
